# Tasks App

Simple task management app to test deploying using vercel

**Live Demo:** https://task-app-frontend-eta.vercel.app/

## Running Frontend

```
cd frontend && npm i
npm run dev
```

-   Runs by default on `http://localhost:3001`

## Running Backend

```
cd backend && npm i
npm start
```

-   Runs by default on `http://localhost:3000`
